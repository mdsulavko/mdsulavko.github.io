THIS IS THE MOST IMPORTANT THING I LEARNED FROM TO ALL OF THIS

Save several backups of every file you create. Physical and cloud copies of a file can be deleted or vanish suddenly so several insurances are needed. I had already made a website that had used databases to save account data, was heavily styled with css, and was very detailed. I had two cloud copies online available, the most important one being on the web server website itself. All cloud copies were found to be deleted when I recently searched and all physical copies are early development stages. Having this project now would've been a big help but I didn't consider the chance of me using it again after I submitted it.

SAVE MANY COPIES OF THE WORK YOU PRODUCE
